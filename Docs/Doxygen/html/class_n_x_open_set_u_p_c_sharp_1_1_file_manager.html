inlinestatic

Ensures that the specified directory exists, creating it if it does not.

If the directory does not exist, this method attempts to create it. If an error occurs during directory creation, the error is logged using the NXLogger instance, if available.

Parameters
directoryThe path of the directory to check and create if it does not exist. Cannot be null or empty.
